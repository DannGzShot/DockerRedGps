# DockerRedGps

Integracion Docker local para REDGPS.

Este repositorio contiene solamente la configuracion de contenedores, scripts auxiliares, plantillas seguras y documentacion para levantar DEV y QA en una computadora local. No incluye los repositorios de aplicacion `dev`, `qa` ni `master`.

Instalacion automatica recomendada:

```bash
make setup-wizard-dry-run
make setup-wizard
```

Guia completa de instalacion automatica, manual y diagnostico:

- [docs/docker-local-redgps.md](docs/docker-local-redgps.md)
